
Welcome to fmanalyser's documentation!
======================================

This module provides tools for working with the
`Pira.cz P175 FM Broadcast Analyser <http://pira.cz/fm_broadcast_analyzer/>`_.

Those tools aim to help using this device in bigger project such as general purpose
monitoring system.
